# Teranga Voyage 🌍

Bienvenue sur le dépôt officiel de Teranga Voyage, votre agence partenaire pour découvrir les merveilles du Sénégal et du monde entier.

 À propos du projet
Ce projet consiste à développer une plateforme moderne et intuitive pour :
* Présenter des destinations touristiques uniques.
* Faciliter la réservation de voyages.
* Promouvoir la culture de la "Teranga" sénégalaise.

 Technologies
*  HTML5, CSS3 boostrap 
*  Git & GitHub
